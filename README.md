# Keep
有时在攻防项目中，在利用漏洞上传webshell或者cs上线后发现被删除了，再想重新上传时发现漏洞被修了，难免口子很可惜就掉了。所以搞了个维持webshell和维持cs online的，免杀测试通过最新版本360以及火绒。（*更新：增加了自定义任务名以及文件隐藏功能）
